    ToolBoxV2¶

    Ein modulares Python-Framework für Tools, Funktionen und komplette Anwendungen — lokal, im Web, als Desktop- oder Mobile-App. 

    pip install toolboxv2 && tb --guide

    ARCHITECTURE¶
    flowchart TD
    Nginx["Nginx<br/>Load Balancing · Rate Limit"]
    Nginx --> W1["HTTP Worker<br/>WSGI"]
    Nginx --> W2["HTTP Worker<br/>WSGI"]
    Nginx --> W3["WS Worker<br/>asyncio"]
    W1 & W2 & W3 --> ZMQ["ZeroMQ Event Broker"]
    ZMQ --> TB["ToolBoxV2 App"]
    TB --> MODS["Mods<br/>CloudM · ISAA · DB · …"]
    TB --> FLOWS["Flows<br/>cli · mini · isaa · desktop"]
    BUILDING BLOCKS¶
    Baustein Beschreibung Docs
    Python Core 4-Layer-Architektur: App, MainTool, Result Core API
    Worker System WSGI/async Worker über ZeroMQ IPC Runtime
    CloudM Auth, User-Daten, Mod-Management, Folder-Sync CloudM
    ISAA Multi-Agent-Framework (FlowAgent, Chains, Skills) ISAA
    tbjs + Tauri Cross-Platform Web-/Desktop-UI tbjs
    TB Registry Package-Registry für Mods und Artefakte Registry
    LLM Gateway OpenAI-kompatibler self-hosted Gateway Gateway
