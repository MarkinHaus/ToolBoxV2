llama_lab API

HomePlaygroundDashboard

OpenAI-compatible API served by local llama.cpp models (text, vision, embeddings and full omni). Point any OpenAI SDK at the base URL below and use your API key from the dashboard.

Onboarding

  1. Get an API key from your dashboard (an admin enables your account).
  2. An admin loads at least one model in the admin panel (browse HuggingFace → download GGUF → load with single or mass mode).
  3. Call the API — endpoints below are drop-in OpenAI compatible.
Base URL: http://<host>:<port>/v1 · Auth header: Authorization: Bearer sk-...

Quickstart (OpenAI SDK)

from openai import OpenAI

client = OpenAI(base_url="http://localhost:8000/v1", api_key="sk-your-key")

resp = client.chat.completions.create(
    model="auto",                       # or a specific loaded model
    messages=[{"role": "user", "content": "Erklär mir MoE in einem Satz."}],
)
print(resp.choices[0].message.content)

Streaming

stream = client.chat.completions.create(
    model="auto", stream=True,
    messages=[{"role": "user", "content": "Zähl bis 5."}],
)
for chunk in stream:
    print(chunk.choices[0].delta.content or "", end="")

Vision / media (model-dependent)

client.chat.completions.create(
    model="auto",
    messages=[{"role":"user","content":[
        {"type":"text","text":"Was ist auf dem Bild?"},
        {"type":"image_url","image_url":{"url":"data:image/png;base64,..."}},
    ]}],
)

The router auto-selects a vision- or omni-capable model when the request contains image/audio parts; otherwise a text model.

Embeddings

client.embeddings.create(model="auto", input=["hello", "world"])

Endpoints

MethodPathPurpose
GET/v1/modelsList loaded models
POST/v1/chat/completionsChat (text/vision/omni, stream)
POST/v1/embeddingsEmbeddings
POST/v1/audio/transcriptionsSpeech-to-text (omni/audio)
POST/v1/audio/liveOpen a live (omni) session → WS

Internal TB client

From inside ToolBoxV2 you can talk to a loaded model without HTTP by using the llama_lab serve layer / the OpenAI base url it exposes:

from toolboxv2 import get_app
app = get_app()

# Which models are currently served?
status = await app.a_run_any(("llama_lab", "status"))

# Start one headlessly (admin/CLI also do this)
await app.a_run_any(("llama_lab", "serve"), name="qwen", mode="single", port=8080)

# Then use any OpenAI client against http://localhost:8080/v1
