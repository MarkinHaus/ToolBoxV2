Getting Started

TB Registry is the package registry for the ToolBox Framework. Install and manage modules using the tb CLI with the CloudM module manager.

Module Management

Interactive Manager

The easiest way to manage modules is the interactive manager:

tb -c CloudM mods manager

Install a Module

tb -c CloudM mods install <module-name>

Or use the shortcut:

tb --install <module-name>

List Modules

tb -c CloudM mods list

Update Modules

tb --update <module-name>

Remove a Module

tb --remove <module-name>

Publishing

Publish your own modules to share with the community. You need to be a verified publisher.

1. Login

tb login

2. Generate Config

Create a tbConfig.yaml for your module:

tb -c CloudM mods gen-config <module-name>

3. Publish

Use the interactive manager to publish:

tb -c CloudM mods manager

API Reference

The registry provides a REST API for programmatic access.

Packages

  • GET /api/v1/packages - List all packages
  • GET /api/v1/packages/{name} - Get package details
  • GET /api/v1/packages/{name}/versions - List versions
  • GET /api/v1/packages/{name}/versions/{v}/download - Get download URL
  • POST /api/v1/packages - Create package (auth)
  • POST /api/v1/packages/{name}/versions - Upload version (auth)

Artifacts

  • GET /api/v1/artifacts - List artifacts
  • GET /api/v1/artifacts/{name} - Get artifact details
  • GET /api/v1/artifacts/{name}/versions/{v}/download - Get download URL

Search & Resolve

  • GET /api/v1/search?q={query} - Search packages
  • POST /api/v1/resolve - Resolve dependencies

Example

curl https://registry.simplecore.app/api/v1/packages/CloudM
